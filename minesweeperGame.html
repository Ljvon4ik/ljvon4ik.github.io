Minesweeper

I've loved this game since childhood and decided to create my own version. The game is currently in development, so you can check the current state of my code on GitHub. The game mechanics are simple, making it convenient to apply and experiment with new knowledge. In addition to using Zenject in this project, I'm also diving deeper into applying SOLID principles. I'm using familiar ScriptableObjects and exploring new features, such as implementing a ScrollView menu without Update using Coroutines.
